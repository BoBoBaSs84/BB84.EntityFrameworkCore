Remarks

This interface extends IGenericRepository<TEntity> and provides additional methods for CRUD operations specifically tailored to entities with identity-based primary keys.

Parameters

fieldSelector Expression<Func<TResult, TResult>>

An optional expression to further select or transform the projected result. If null, the entire projection defined by selector is returned.

ignoreQueryFilters bool

true to ignore any global query filters applied to the entity type; otherwise, false.

token CancellationToken

A cancellation token that can be used to cancel the asynchronous operation.

Returns

Task<TResult>

A task that represents the asynchronous operation. The task result contains the projected entity if found; otherwise, null.

Type Parameters

TResult

The type of the result returned by the selector expression.

Remarks

Use this method to efficiently retrieve and project a single entity by its key, minimizing data transfer by selecting only required fields. If the entity is not found, the result is null. When ignoreQueryFilters is set to true, any global query filters (such as soft-delete or multi-tenancy filters) are bypassed for this query.
